ONE-ARMED BASTION — прототип
============================

prototype.html     — играбельный прототип (открой в браузере)
concept.md         — концепция игры
slot-machine.html  — отдельный «однорукий бандит» с таблицей выплат

Как запустить: открой prototype.html двойным кликом в любом браузере.
Интернет нужен только для подгрузки шрифтов (оформление), сама игра работает офлайн.
